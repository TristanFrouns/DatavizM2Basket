Dataset:

Ces Diagrammes ont été réalisés à partir de l'outil Query SPARQL du site WikiData et ont été transformés à l'aide de CHAT GPT4.

Voici les requette SPARQL

SELECT ?National_Basketball_AssociationLabel ?country_of_citizenshipLabel ?work_period__start_Label WHERE { SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". } ?National_Basketball_Association wdt:P118 wd:Q155223. ?National_Basketball_Association wdt:P27 ?country_of_citizenship. ?National_Basketball_Association wdt:P2031 ?work_period__start_. } LIMIT 10000
