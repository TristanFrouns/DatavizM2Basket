Exercice n.o 01

Coparaison des occupations des gens entérrés dans le cimetierre du Père lachaise

Ce diagramme montre les rapports de quatité qui existent entre les métiers des personnes entérrées au P. Lachaise

la requète est :

SELECT ?Père_Lachaise_Cemetery ?award_received ?award_receivedLabel ?given_name ?given_nameLabel ?family_name ?family_nameLabel WHERE { SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". } OPTIONAL { } ?Père_Lachaise_Cemetery wdt:P119 wd:Q311. OPTIONAL { ?Père_Lachaise_Cemetery wdt:P735 ?given_name. } OPTIONAL { ?Père_Lachaise_Cemetery wdt:P734 ?family_name. } OPTIONAL { } } LIMIT 10000
